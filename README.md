# NUPAT AI Fellowship – Stage Two Case Study

This repository contains a single Jupyter Notebook submitted as part of the
NUPAT AI Fellowship Stage Two assessment.

## Contents
- `NUPAT.ipynb` – Complete analysis notebook
- `trades.csv` – Trading activity dataset
- `user_activity.csv` – User deposit and withdrawal dataset

## Overview
The notebook analyzes user trading and activity data to:
- Explore market dynamics and user behavior
- Identify potentially suspicious user activity using a rule-based and
  machine learning approach
- Provide a data-driven strategic recommendation

## How to Run
Open the notebook and run all cells from top to bottom.
All required data files are included in the repository.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Scikit-learn
